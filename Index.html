Arithmos

How To Play

  • Each clue cell contains two numbers.
  • Top-right number → Horizontal (Across) sum.
  • Bottom-left number → Vertical (Down) sum.
  • Numbers 1–9 only.
  • No duplicate number allowed in any run.
  • All white cells must be filled.

Controls

  • Click a cell to select it.
  • Use number pad or keyboard (1–9).
  • Undo button reverses last move.
  • Clear button resets board.

Competition Rules

  • Unlimited wrong attempts allowed.
  • Timer starts when game begins.
  • After correct submission, board locks.
  • 52-character secure code generated.
